# 🤖 RoboVision AI

> AI-powered Robotics Inspection using Computer Vision and Large Language Models.

---

## 📌 Overview

RoboVision AI is an AI-powered application that analyzes robotics project images and automatically generates a structured engineering inspection report.

The application combines BLIP for image captioning and Qwen2.5 for technical reasoning to generate reliable engineering reports based only on the visible components in the uploaded image.

---

## ✨ Features

- 🤖 Robotics Project Analysis
- 📷 Image Captioning using BLIP
- 🧠 Engineering Report Generation using Qwen2.5
- 🔧 Visible Components Detection
- 📡 Visible Sensors Identification
- ⚙️ Controller Board Recognition
- 🔋 Power Source Identification
- 🚀 Engineering Improvement Suggestions
- 📋 Bill of Materials (BOM)
- 🌐 Interactive Gradio Interface
